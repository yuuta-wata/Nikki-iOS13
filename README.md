# Nikki-iOS13
ローカルに保存ができる日記アプリです📔

## アプリケーション機能
・📮記事投稿機能  

・📄記事一覧表示機能  

・🗓カレンダーの日付毎に記事一覧表示する機能  

・📝記事編集機能  

・🗑記事削除機能  

## 使用ライブラリ
RealmSwift  

FSCalendar
